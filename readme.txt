# 🎬 JetFilim - Profesyonel Film Streaming Sitesi

Modern ve kullanıcı dostu film streaming platformu. Netflix benzeri tasarım ve gelişmiş özelliklerle.

## ✨ Özellikler

### 🎥 **Film & Dizi Yönetimi**
- Drag & Drop ile video dosyası yükleme
- Film/dizi ekleme, düzenleme, silme
- Kategori ve tür filtreleme
- IMDB puanı ve süre bilgisi
- Poster yükleme desteği

### 🔐 **Admin Panel**
- **Gizli Erişim**: `Shift + F12` tuş kombinasyonu
- Tam CRUD işlemleri
- Kullanıcı dostu arayüz
- Responsive tasarım

### 🎨 **Modern Tasarım**
- Netflix benzeri kullanıcı arayüzü
- Smooth animasyonlar ve hover efektleri
- Glassmorphism ve gradient efektler
- Koyu tema optimizasyonu
- Tam responsive tasarım

### 🔍 **Gelişmiş Arama**
- Gerçek zamanlı arama
- Kategori filtreleme
- Tür bazlı filtreleme
- Sonuç sayısı gösterimi

### 📱 **Responsive Tasarım**
- Mobil, tablet ve desktop uyumlu
- Adaptive grid sistemi
- Touch-friendly arayüz

## 🚀 Kurulum

### Gereksinimler
- Node.js 18+
- npm veya yarn

### Adımlar
```bash
# Projeyi klonla
git clone [proje-url]
cd jetfilim

# Bağımlılıkları yükle
npm install

# Geliştirme sunucusunu başlat
npm run dev

# Production build
npm run build
```
